# Template Repository Reference

Curated collection of the best open-source website template repositories (2026).

## SaaS & Startup Landing Pages

### MasuRii ModernSaaS
- **URL**: https://github.com/MasuRii/ModernSaaS-LandingPage-Template
- **Stack**: Astro 5, React 19, Tailwind CSS v4, Motion One
- **Features**: Bento grids, glassmorphism, mesh gradients, near-zero JS by default
- **Best for**: Tech/SaaS landing pages with modern visual effects

### Orbitly Template (Open Antimatter)
- **URL**: https://github.com/mattantimatter/orbitly-template
- **Stack**: React 19, Vite 7, Tailwind CSS 4, Framer Motion 12
- **Features**: Scroll-aware glass blur navbar, animated logo marquees, masonry testimonials
- **Best for**: Startup product pages with rich animations

### Riccoc SaaS Template
- **URL**: https://github.com/ricocc/ricoui-saas-template
- **Stack**: Astro 6, Tailwind CSS v4
- **Features**: 11 production-ready pages (Pricing, Blog, Changelog, Auth), Lighthouse 95+
- **Best for**: Full multi-page SaaS sites

---

## Multi-Purpose & Agency Kits

### TrinkUI
- **URL**: https://github.com/trinkui/trinkui
- **Stack**: React, Tailwind v4, TypeScript
- **Features**: 8 templates (SaaS, Agency, AI/ML, Mobile App, Startup, Portfolio, Event, Product Launch), 49+ tree-shakeable components
- **Best for**: Agencies needing variety

### DevKit Landing Page Kit
- **URL**: https://github.com/niks-nikhil-anand/landing-page-kit-Devkit
- **Stack**: Next.js 14/15, Tailwind CSS, shadcn/ui
- **Features**: Centralized config file, built-in waitlist with Zod validation
- **Best for**: Fast-deploy landing pages with email collection

---

## AI-Native Generators

### OpenPage
- **URL**: https://github.com/buildingopen/openpage
- **Stack**: React 19, Tailwind CSS v4, TypeScript
- **Features**: Visual drag-and-drop editor, AI generation via natural language, JSON-first architecture (Git-friendly), one-click HTML/React export
- **Best for**: Teams wanting visual editing + version control

### Open Builder
- **URL**: https://github.com/Amery2010/open-builder
- **Stack**: 20+ frameworks (Astro, Svelte, SolidJS, React)
- **Features**: Natural language → full project, Sandpack real-time preview, smart file patching
- **Best for**: Rapid prototyping across frameworks

### Promter
- **URL**: https://github.com/edensitko/Promter
- **Stack**: Prompt-based, framework-agnostic
- **Features**: 27+ pre-built prompt templates (SaaS, e-commerce, dashboards), UI styles (Glassmorphism, Brutalist), battle-tested AI prompts
- **Best for**: Getting high-quality UI from AI without generic look

---

## Documentation & Content Sites

### Astropress
- **URL**: https://github.com/Astropress/astropress
- **Stack**: Static/Dynamic (GitHub Pages, Cloudflare, Vercel)
- **Features**: SEO tools, media optimization, multilingual, WCAG 2.2 AA compliant
- **Best for**: Content sites and blogs needing accessibility

### Starlight (Astro)
- **URL**: https://starlight.astro.build
- **Stack**: Astro
- **Features**: Built-in search, sidebar nav, code highlighting, i18n
- **Best for**: Open-source documentation sites

---

## Dashboards

### Horizon UI
- **URL**: https://horizon-ui.com
- **Stack**: Next.js / Tailwind HTML
- **Features**: Admin panels, SaaS dashboards, data-heavy UIs
- **Best for**: Internal tools and analytics dashboards

## Selection Guide

| If user wants... | Recommend |
|---|---|
| Quick landing page | MasuRii ModernSaaS or Orbitly |
| Full SaaS site | Riccoc SaaS Template |
| Agency/portfolio suite | TrinkUI |
| AI-generated from prompt | OpenPage or Open Builder |
| High-end AI prompts | Promter |
| Documentation | Starlight |
| Dashboard | Horizon UI |
| Accessibility-first | Astropress |

## Key Design Trends (2026)

- **Bento Grids**: Asymmetrical, rounded feature boxes (Apple/Stripe style)
- **Glassmorphism**: Frosted glass effects on blurred backgrounds
- **Scroll-triggered animations**: Staggered reveals, parallax
- **Fluid typography**: clamp()-based responsive sizing
- **Container queries**: Component-based responsive design
- **Dark mode as default**: Most templates ship dark-first with light option
